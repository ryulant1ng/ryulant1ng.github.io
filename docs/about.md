### Typographical conventions

#### General

- Capitalize only the first letter of the title
- Italics are used for emphasize a word in a text, for emphasize a sentence, use bold (including the punctuation at the end of it)
- When listing three or more items, add 'and' before the last item and a comma before 'and'
- Add a dagger (†) before dead people, and write their date of birth and death in the first time appear

#### CJK people names

- **China**: Lastname, firstname
    - **My name is the exception, it will be always write as 'Lanting Liu'**
    - e.g. †Jiang Zemin (17 August 1926 - 30 November 2022)
- **Macau, North Korean, South Korean**: Lastname, firstname
    - Write each word separately
    - e.g. Kim Jong Un
- **Hong Kong**: English given name (if have), lastname, firstname
    - Write each firstname word separately, connect with '-'
    - e.g. Joshua Wong Chi-fung
- **Taiwan**: Lastname, firstname
    - Write each firstname word separately, connect with '-'
    - e.g. Tsai Ing-wen
- **Japan**: Firstname, lastname
    - e.g. †Shinzo Abe (21 September 1954 - 8 July 2022)
- **Old names**: respect the prevailing spelling
    - e.g. Sun Yat-sen

#### Lists

- For every list item, don't write period
- Make each item's subtitle bold (before the colon) and capitalize the first letter of each sentence after the colon
- Don't use list with one item

#### Quotes

- Use block quotes whenever possible, a vaild block quote is like this:

> (quoted content)
> 
> ― Name, _Book Name_ (if have)

- The dash before name is '―' (U+2015, quotation dash)

#### Dashes

Following dashes is used:

- '—' (U+2014, em dash)
- '―' (U+2015, quotation dash)
- ' - ' (one space on each side of the hyphen, as en dash)
